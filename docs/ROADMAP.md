# Roadmap

Starter roadmap:

1. Build the polished React Native Expo prototype.
2. Finalize the Home screen and Wisdom Detail screen.
3. Build reading and read-to-me experiences.
4. Add AI conversation with Cloud.
5. Add short quizzes and real-life challenges.
6. Add favorites and growth milestones.
7. Add parent dashboard.
8. Connect backend, data, auth, unlock logic, and analytics.

Do not add backend or business logic before the UI direction is approved.

Do not redesign product direction without approval.
