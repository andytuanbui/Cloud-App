Luma — Visual Direction Test

How to use this during a session
  1. Labels are hidden by default so the first look is blind, as the testing plan calls for. Keep it that way while you ask "which would you tap on / which would you want to open?"
  2. Use Swap sides between participants so the same version isn't always on the left — guards against a left-side bias in answers.
  3. Use the tabs to move through the three key screens once initial reactions are captured, then open the Streak Probe tab for Step 5's optional counter-vs-streak question.
  4. Tap Reveal labels only after you've captured reactions, to confirm which version the participant preferred.
  5. This file works offline once loaded — open it on a phone or tablet and hand it over, or share your screen.
